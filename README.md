# Objective

Build off of Lab 1 and create a HTML document that shows butterfly images and shows a search box to search for butterflies by name along with input validation

# Features

- Added a set of checkboxes for butterfly type next to the search box. Ensures that the checkboxes are all checked by default.
  
- Whenever a checkbox state is changed, ensures that search results only include butterfly types with a check mark.

- A new list inside this block with list elements for all of the search results. Each list element has all the information that is present in the full list.
  
- Dynamically updates this list as the user is typing new characters or deleting characters in the search box or changing state of checkboxes.
